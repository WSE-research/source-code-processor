# Method information retrieval


## Usage

### Include as dependency

```
<dependency>
  <groupId>org.wseresearch</groupId>
  <artifactId>processor</artifactId>
  <version>0.0.1</version>
</dependency>
```

### Include to maven-compiler-plugin as annotationPath

```
<path>
  <groupId>org.wseresearch</groupId>
  <artifactId>processor</artifactId>
  <version>0.0.1</version>
</path>
```
